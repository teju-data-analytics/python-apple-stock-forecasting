# Apple Stock Forecasting using Python

Author: Maria Tejaswini Polishetty

## Project Overview
This project analyzes historical Apple Inc. stock data and builds predictive models to forecast future stock performance. The analysis follows the OSEMiN framework and applies time series forecasting and machine learning techniques.

## Objectives
- Forecast Apple stock prices using ARIMA models
- Compare predictive accuracy with Random Forest models
- Analyze stock trends, volatility, and market behavior

## Dataset
The dataset contains historical Apple stock prices including:
- Date
- Open
- High
- Low
- Close
- Adjusted Close
- Volume

Additional variables such as returns, moving averages, and rolling volatility were created during preprocessing.

## Methods Used
- Exploratory Data Analysis (EDA)
- Time Series Analysis
- ARIMA Forecasting
- Random Forest Machine Learning
- K-Means Clustering
- A/B Strategy Testing

## Key Findings
Random Forest achieved the lowest RMSE (81.55) compared to ARIMA (91.74), indicating better predictive performance.
